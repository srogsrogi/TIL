# DNS

- IPv4 주소로 어떻게 접속까지는 했는데..
- 주소가 너무 못 생겼다.
- 나는 주소에 `naver.com` 처럼 예쁜 이름을 달아주고 싶어! -> **Domain Name System**



##### 근데.. DNS 자체는 얕게만 알아도 당장 쓰는데 큰 무리가 없어서, 개념 설명은 짧고..
